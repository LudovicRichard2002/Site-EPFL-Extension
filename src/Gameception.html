What is GameCeption.

See More Features

GameCeption is a 3D Standalone development kit. Fueled up by hundreds of free assets allowing whoever to create their own game. Create your own map without modeling and your own game system without programming only by changing prefab parameters. Export your project, import it on Unreal Engine 5, Package it. Sell it, Share it , Play it.

Inside Tools.

Item Browser Tool

Search up among hundreds of 3D assets, light effects, animations and LUTs. Place any items by drag and dropping them directly in the 3D scene. Edit them using the Gizmo Tool.

Read More

Lighting Tool

Create any light effect using the templates. Change the emission type and the intensity. It includes global illumination and Ray Tracing.

Read More

Landcaping Tool

Edit the shape of your root terrain using the Landscaping Tool.

Read More

Programming Tool

Change variables value presented as "parameters" using a convenient user interface. Use "trigger volumes" to trigger action by player location.

Read More

AI Path System Tool

Define your AI behaviour and path using the Path System Tool. Trace their way to follow and define action by location..

Read More

Post Process Tool

Change the tint and saturation in post prod. Use the Post Process Volume to define any place of any size where post process applies.

Some Features

VR Ready

Use the VR template to create your virtial reality project. Working with any headset using SteamVR or Microsoft.

Multiplayer Development

Make your project with up to 4 users on the same server ! Save locally on every machine.

Undreds of Assets included

Enjoy a complete library full of modulables PBR assets and materials. Only your imagination is the limit.

Inside Modeling

Use the Gizmo Tool to resize your meshes. Create some of your own using the Modeling or the Procedural Tool.

Editable AI

Managing your AI has never been that simple! Define theirs settings as their field of view, their health points, their position etc... without programming!

7/7 Support

You can get support from installation to packaging your project at anytime by E-mail using the contact form.
